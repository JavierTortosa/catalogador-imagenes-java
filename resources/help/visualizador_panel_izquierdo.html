El Panel de Navegación Izquierdo

Este panel es tu centro de control para encontrar y filtrar las imágenes que quieres visualizar. Está organizado en tres pestañas con funcionalidades muy potentes.

Pestaña Lista

Muestra una lista con los nombres de archivo de las imágenes en la carpeta actual. La barra de herramientas superior te permite navegar y filtrar esta lista.

Navegar a Carpeta Raíz: Vuelve a la carpeta principal que abriste inicialmente.
Subir un Nivel: Navega a la carpeta contenedora de la actual, guardando el historial para poder regresar.
Entrar en Subcarpeta: Si tienes una subcarpeta seleccionada en la lista, este botón te permite navegar dentro de ella, guardando el historial.
Ciclo de Orden: Cambia el orden de los archivos (por defecto, A-Z, Z-A).
Filtro "Tornado" (Filtro en Vivo): Cuando está activo, el campo de texto filtra la lista de archivos en tiempo real mientras escribes. Es una forma muy rápida de encontrar algo. Si está desactivado, el campo de texto sirve para buscar la siguiente coincidencia al pulsar Enter.
Hacer Filtro Persistente: Toma el texto del filtro "Tornado" y lo añade como un filtro permanente en la pestaña "Filtros". Útil cuando un filtro rápido te resulta muy útil y quieres guardarlo.

Pestaña Carpetas

Muestra una vista de árbol de tu sistema de archivos. Esta pestaña tiene un sistema de navegación con historial:

  • Doble Clic (o "Abrir aquí" con clic derecho): Carga la carpeta seleccionada como la nueva carpeta raíz. Esto limpia el historial de navegación y establece un nuevo punto de partida. Es como usar "Archivo > Abrir Carpeta".
  • Clic Simple (o "Entrar en esta carpeta" con clic derecho): Entra en la carpeta de forma temporal, guardando tu ubicación anterior en el historial. Puedes volver a la carpeta anterior usando el botón "Subir un Nivel" de la Pestaña Lista.

Pestaña Filtros

Aquí puedes construir filtros complejos y persistentes para mostrar solo las imágenes que te interesan. Puedes combinar múltiples reglas.

/ Fuente del Filtro: Elige si el texto que escribas se buscará en el nombre del archivo o en la ruta de la carpeta.
/ Tipo de Filtro: Añade un filtro positivo [+] (la imagen DEBE contener el texto) o un filtro negativo [-] (la imagen NO DEBE contener el texto).
Eliminar Filtro: Quita la regla de filtro que tengas seleccionada en la lista de filtros activos.
Limpiar Todos: Elimina todas las reglas de filtro activas y vuelve a mostrar la lista completa de archivos.
