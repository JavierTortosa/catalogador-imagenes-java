Componentes del Modo Proyecto

El Modo Proyecto es tu espacio de trabajo dedicado. Aquí puedes gestionar, refinar y exportar las colecciones de imágenes que has marcado previamente.

Conceptos Clave

  • Selección: Es la lista principal de imágenes que has marcado porque te interesan. Esta es la lista que normalmente exportarás.
  • Descartes: Es una lista temporal para imágenes que marcaste, pero que después de una revisión has decidido que no te sirven para este proyecto. Funciona como una "papelera de reciclaje" interna del proyecto.
  • Archivo de Proyecto (.prj): Es el archivo físico en tu disco duro donde se guarda tu lista de Selección y Descartes. Te permite cerrar la aplicación y continuar tu trabajo más tarde.

La Interfaz

El modo proyecto tiene una estructura diferente:

  • Panel de Listas: A la izquierda, verás la lista de Selección. Haciendo clic en la pestaña inferior, puedes cambiar a la vista de Descartes.
  • Área de Visualización: A la derecha, puedes ver las imágenes de la lista activa (Selección o Descartes), ya sea una a una o en una vista de cuadrícula (Grid).
  • Panel de Herramientas: Debajo de la lista de Selección, encontrarás herramientas avanzadas como el potente sistema de Exportación.
