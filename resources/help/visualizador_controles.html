Controles del Modo Visualizador

Modos de Visualización de Contenido

El visualizador puede presentar el contenido de dos formas principales, que se seleccionan desde la barra de herramientas "Vista" o el menú Vista > Modos de Visualización.

Vista de Imagen Única

El modo por defecto. Muestra una sola imagen grande en el centro, con la lista de nombres a la izquierda y la barra de miniaturas abajo. Ideal para inspeccionar imágenes en detalle.
Vista de Cuadrícula (Grid)

Oculta la barra de miniaturas y transforma el visor principal en una cuadrícula de imágenes. Es útil para tener una vista general de muchas imágenes a la vez. Puedes hacer doble clic en una miniatura para abrir una previsualización grande.

Barras de Herramientas Principales

Navegación y Orden

Estos controles te permiten moverte por la lista de imágenes y cambiar su orden.

/ Anterior / Siguiente: Navega a la imagen previa o siguiente en la lista. También funciona con las flechas del teclado y la rueda del ratón.
Navegación por Carpetas: Permite entrar y salir de subcarpetas o volver a la carpeta raíz sin usar el menú "Archivo".
Ciclo de Orden: Cambia el orden de la lista de archivos (Apagado -> Ascendente A-Z -> Descendente Z-A).

Zoom

Controla cómo se muestra la imagen en el visor principal.

Modos de Zoom: Un grupo de botones para ajustar automáticamente el tamaño de la imagen (Ajustar a Pantalla, 100%, Ajustar a Ancho, etc.).
Activar Zoom Manual: Permite usar la rueda del ratón para hacer zoom y arrastrar la imagen para panear. Cuando está desactivado, la rueda navega entre imágenes.

Toggles de Comportamiento

Estos botones activan o desactivan funcionalidades clave.

Incluir Subcarpetas: Si está activo, la lista de archivos mostrará las imágenes de la carpeta actual y de todas sus subcarpetas.
Mantener Proporciones: Si está activo, las imágenes se ajustarán al visor sin deformarse. Si se desactiva, se estirarán para rellenar todo el espacio.
Marcar para Proyecto (Barra Espaciadora): Añade o quita la imagen actual de la "Selección Actual" del Modo Proyecto.
