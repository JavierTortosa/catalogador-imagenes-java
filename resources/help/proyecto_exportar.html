Guía del Panel de Exportación

El Panel de Exportación es la herramienta para recopilar y organizar las imágenes de tu "Selección Actual" junto con sus archivos comprimidos asociados (.zip, .rar, etc.) para copiarlos a una carpeta de destino.

Anatomía del Panel

El panel se divide en tres áreas funcionales:

1. La Tabla de Exportación

Es el componente central. Cuando se abre el panel, se puebla con las imágenes de tu "Selección Actual" y la aplicación busca automáticamente sus archivos asociados.

Columna Descripción
✔ (Checkbox) Permite seleccionar o deseleccionar un ítem para incluirlo en la exportación. Puedes marcar/desmarcar todos haciendo clic en la cabecera de la columna.
Imagen Muestra el nombre del archivo de la imagen.
Estado La columna más importante. Muestra con iconos y colores si se ha encontrado el archivo comprimido. Consulta la sección de estados más abajo.
Archivos Asignados Lista los nombres de los archivos comprimidos que se copiarán. Si hay más de un archivo asignado, aparecerá un icono especial que te invita a abrir el Panel de Detalles.

Nota: Puedes hacer clic derecho sobre una fila de la tabla para acceder a un menú contextual con acciones rápidas.

2. El Panel de Detalles

Ubicado debajo de la tabla, este panel muestra en detalle la lista de rutas completas de los archivos comprimidos asignados. Para mostrarlo, pulsa el botón en la barra de acciones.

¡Truco! Cuando seleccionas una fila en la tabla con múltiples archivos asignados, el botón para mostrar los detalles en la barra de acciones se resaltará con un marco para indicarte que es relevante.

Este panel tiene su propia barra de botones para gestionar los archivos de la lista:

Añadir: Permite asignar manualmente uno o más archivos comprimidos adicionales.
Quitar: Elimina un archivo asociado de la lista (no borra el archivo del disco).
Localizar: Abre el explorador de archivos en la ubicación del archivo asociado seleccionado.

3. La Barra de Acciones (Derecha)

Esta barra vertical contiene todos los controles para gestionar la cola de exportación:

Seleccionar Carpeta: Abre un diálogo para elegir la carpeta de destino (obligatorio para exportar). El campo de texto inferior mostrará la ruta seleccionada.
Mostrar/Ocultar Detalles: Alterna la visibilidad del panel de detalles inferior.
Asignar Archivo: Permite localizar manualmente el/los archivo(s) comprimido(s) si la aplicación no lo encontró.
Ignorar Comprimido: Marca un archivo como "listo" aunque no tenga archivo asociado. Se exportará solo la imagen.
Relocalizar Imagen: Si una imagen del proyecto fue movida, permite encontrarla en su nueva ruta.
Refrescar Estado: Vuelve a escanear el disco para actualizar el estado de todos los archivos.
Mover a Descartes: Quita la imagen de la cola de exportación y la mueve a "Descartes".
Iniciar Exportación: Comienza la copia. Solo se activa cuando hay una carpeta de destino y todos los ítems marcados están listos.

Entendiendo la Columna 'Estado'

El estado de cada ítem determina si puede ser exportado. Aquí tienes la guía de colores y significados:

Estado Significado y Acción Requerida
✔ ENCONTRADO_OK ¡Perfecto! La aplicación ha encontrado el archivo asociado. No se requiere acción.
ℹ ASIGNADO_MANUAL Has asignado manualmente un archivo. El ítem está listo.
✔ IGNORAR_COMPRIMIDO Has decidido exportar esta imagen sin archivo comprimido. El ítem está listo.
⚠ NO_ENCONTRADO No se encontró archivo asociado. Debes asignarlo manualmente o ignorarlo.
❌ IMAGEN_NO_ENCONTRADA ¡Error Crítico! El archivo de la imagen no se encuentra. Usa "Relocalizar Imagen".
