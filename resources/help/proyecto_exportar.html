El Panel de Exportación

El Panel de Exportación es la herramienta principal para recopilar y organizar las imágenes de tu proyecto junto con sus archivos comprimidos asociados (como .zip o .rar) para su fácil distribución o respaldo.

Anatomía del Panel

El panel se divide en tres áreas funcionales:

1. La Tabla de Exportación

Es el componente central donde se gestiona la lista de archivos. Cada columna tiene un propósito específico:

Columna Descripción
✔ (Checkbox) Permite seleccionar o deseleccionar un ítem para incluirlo en la exportación. Puedes marcar/desmarcar todos haciendo clic en la cabecera de la columna.
Imagen Muestra el nombre del archivo de la imagen. Puedes hacer clic en la cabecera para ordenar la lista alfabéticamente.
Estado La columna más importante. Muestra con iconos y colores si se ha encontrado el archivo comprimido asociado. Consulta la sección de estados más abajo para una explicación detallada.
Archivos Asignados Lista los nombres de los archivos comprimidos que se copiarán junto con la imagen. Si no se encontró ninguno, mostrará "Ninguno".

Nota: Puedes hacer clic derecho sobre una fila de la tabla para acceder a un menú contextual con acciones rápidas.

2. El Panel de Detalles

Ubicado debajo de la tabla (puedes mostrarlo u ocultarlo), este panel muestra en detalle la lista de rutas completas de los archivos comprimidos asignados al ítem seleccionado. Cuenta con su propia barra de botones:

Añadir: Permite asignar manualmente uno o más archivos comprimidos a la imagen seleccionada.
Quitar: Elimina un archivo asociado de la lista (no borra el archivo del disco).
Localizar: Abre el explorador de archivos en la ubicación del archivo asociado seleccionado.

3. La Barra de Acciones Inferior

Contiene los controles finales para el proceso de exportación:

Seleccionar Carpeta: Abre un diálogo para elegir la carpeta de destino obligatoria para la exportación.
Mostrar/Ocultar Detalles: Alterna la visibilidad del panel de detalles inferior.
Asignar Archivo: Permite localizar manualmente el archivo comprimido (.rar, .zip) si la aplicación no lo encontró automáticamente.
Ignorar Comprimido: Marca un archivo como "listo" para exportar aunque no se haya encontrado su .rar/.zip asociado.
Relocalizar Imagen: Si una imagen del proyecto fue movida de su ubicación original, esta opción permite encontrarla en su nueva ruta.
Refrescar Estado: Vuelve a escanear el disco para actualizar el estado de todos los archivos de la tabla. Útil si has hecho cambios fuera de la aplicación.
Mover a Descartes: Quita la imagen seleccionada de la cola de exportación y la mueve a la lista de "Descartes".
Iniciar Exportación: Comienza el proceso de copia. Este botón solo se activará cuando todos los ítems marcados estén listos y se haya seleccionado una carpeta de destino.

Entendiendo la Columna 'Estado'

El estado de cada ítem determina si puede ser exportado correctamente. Aquí tienes una guía de los posibles estados:

Estado Significado y Acción Requerida
✔ ENCONTRADO_OK ¡Perfecto! La aplicación ha encontrado automáticamente uno o más archivos comprimidos asociados. No se requiere ninguna acción.
⚠ NO_ENCONTRADO La aplicación no encontró ningún archivo comprimido que coincida. Debes asignarlo manualmente o marcarlo para ignorar.
ℹ ASIGNADO_MANUAL Has asignado manualmente un archivo a esta imagen. El ítem está listo para ser exportado.
✔ IGNORAR_COMPRIMIDO Has decidido exportar esta imagen sin ningún archivo comprimido asociado. El ítem está listo para ser exportado.
❌ IMAGEN_NO_ENCONTRADA ¡Error Crítico! El archivo de la imagen original no se encuentra en el disco. Debes usar la opción "Relocalizar Imagen" para buscar su nueva ubicación.
