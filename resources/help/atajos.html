Atajos de Teclado

Usa estos atajos para trabajar de forma más rápida y eficiente.

Navegación General

  • Flechas / Rueda Ratón: Imagen Siguiente/Anterior
  • Inicio / Fin: Primera / Última Imagen
  • Av Pág / Re Pág: Saltar bloque de imágenes

Gestión de Proyectos y Archivos

  • Barra Espaciadora: Marcar / Desmarcar imagen para el proyecto
  • Ctrl + L: Localizar archivo actual en el explorador

Ventana y Vista Global

  • F5: Refrescar lista de imágenes
  • F11: Activar / Desactivar Modo Pantalla Completa
  • Alt: Activar Foco en la Barra de Menú
  • ESC: Cerrar Ventana de Previsualización (si está abierta)

Zoom y Paneo (Sobre la imagen)

  • Rueda Ratón: Zoom (con modo zoom manual activo)
  • Shift + Rueda Ratón: Paneo Horizontal rápido
  • Ctrl + Rueda Ratón: Paneo Vertical rápido
  • Clic y Arrastrar: Panear imagen

Atajos Numéricos para Modos de Zoom:

  • 1: Ajustar a Pantalla
  • 2: Tamaño Original (100%)
  • 3: Ajustar a Ancho
  • 4: Ajustar a Alto
  • 5: Rellenar Pantalla
  • 6: Mantener Zoom Actual (Fijo)
  • 7: Zoom Personalizado (%)
  • 8: Activar/Desactivar Zoom/Paneo Manual
  • 9: Resetear Zoom

Atajos del Grid (Modo Proyecto)

  • Ctrl + T: Añadir / Editar Etiqueta
  • Ctrl + Supr: Borrar Etiqueta
  • Ctrl + Más (+): Aumentar Tamaño de Miniaturas
  • Ctrl + Menos (-): Reducir Tamaño de Miniaturas
