Controles del Modo Proyecto

La interfaz del Modo Proyecto está dividida en las listas de trabajo y un panel de herramientas con pestañas.

Listas de Imágenes

A la izquierda se muestran las dos listas principales. La lista que tiene el foco (con el fondo más claro) es la que está activa para la navegación y las acciones.

  • Selección Actual: Contiene las imágenes marcadas. Sobre estas imágenes se aplican las acciones de las pestañas "Exportar" y "Etiquetar".
  • Descartes: Contiene las imágenes que has apartado temporalmente de la selección principal.

Acciones sobre las Listas

Mover a Descartes / Restaurar a Selección: Este botón cambia su función dependiendo de la lista activa. Si tienes una imagen seleccionada en "Selección Actual", la moverá a "Descartes". Si la tienes en "Descartes", la restaurará a "Selección Actual". El atajo de teclado es la barra espaciadora.

Panel de Herramientas

A la derecha se encuentran las pestañas con las herramientas disponibles.

Pestaña Descartes

Actualmente, las acciones sobre los elementos de la lista de Descartes (como restaurar o eliminar) se realizan a través del botón principal de la barra de herramientas y futuros menús contextuales.

Pestaña Exportar

Prepara las imágenes de la "Selección Actual" para ser copiadas a otra carpeta. La barra de herramientas inferior contiene las siguientes acciones:

Seleccionar Carpeta: Abre un diálogo para elegir la carpeta de destino para la exportación.
Quitar de la cola: Elimina el ítem seleccionado de la tabla de exportación.
Asignar archivo: Permite localizar manualmente el archivo comprimido (.rar, .zip) si la aplicación no lo encontró automáticamente.
Ignorar comprimido: Marca un archivo como "listo" para exportar aunque no se haya encontrado su .rar/.zip asociado.
Relocalizar Imagen: Si una imagen del proyecto fue movida de su ubicación original, esta opción permite encontrarla en su nueva ruta.
Iniciar Exportación (Rayo): Comienza el proceso de copia de los archivos válidos y seleccionados a la carpeta de destino.

Pestaña Etiquetar

Herramientas para gestionar las etiquetas de las imágenes en la vista de Grid.

Etiquetar (Ctrl+T): Abre un diálogo para añadir o editar la etiqueta de la imagen seleccionada en el grid.
Borrar Etiqueta (Ctrl+Supr): Elimina la etiqueta de la imagen seleccionada.
Aumentar Tamaño (Ctrl + '+'): Hace las miniaturas del grid más grandes.
Reducir Tamaño (Ctrl + '-'): Hace las miniaturas del grid más pequeñas.
Mostrar Estado: Activa o desactiva el resaltado de color (verde/amarillo/rojo) en las celdas del grid para reflejar el estado de exportación.
