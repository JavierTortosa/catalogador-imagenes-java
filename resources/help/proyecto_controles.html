Controles y Acciones del Modo Proyecto

El Modo Proyecto cuenta con barras de herramientas específicas para gestionar tus colecciones de forma eficiente.

Barra de Herramientas Principal (Superior)

Esta barra contiene los controles de gestión del archivo de proyecto.

Mover a Descartes / Restaurar: (Atajo: Barra Espaciadora). Si una imagen está en "Selección Actual", la mueve a "Descartes". Si está en "Descartes", la restaura a "Selección Actual".
Nuevo Proyecto (Ctrl+N): Cierra el proyecto actual y crea uno nuevo, vacío y temporal.
Abrir Proyecto (Ctrl+O): Abre un diálogo para cargar un archivo de proyecto .prj existente.
Guardar Proyecto (Ctrl+S): Guarda los cambios en el archivo .prj actual. Si es un proyecto temporal, se comportará como "Guardar Como...".
Guardar Proyecto Como (Ctrl+Shift+S): Guarda el proyecto actual en un nuevo archivo .prj.
Mostrar/Ocultar Panel de Herramientas: Alterna la visibilidad del panel inferior derecho que contiene las pestañas de Exportar y Etiquetar.
Eliminar Proyecto: (Funcionalidad futura) Permitirá eliminar el archivo .prj del disco.

Herramientas del Grid

Cuando estás en la vista de cuadrícula (Grid), una barra de herramientas especial aparece para gestionar las etiquetas.

Etiquetar (Ctrl+T): Abre un diálogo para añadir o editar la etiqueta de la imagen seleccionada.
Borrar Etiqueta (Ctrl+Supr): Elimina la etiqueta de la imagen seleccionada.
Aumentar Tamaño (Ctrl + '+'): Hace las miniaturas del grid más grandes.
Reducir Tamaño (Ctrl + '-'): Hace las miniaturas del grid más pequeñas.
Mostrar Estado: Activa o desactiva un borde de color en las miniaturas (verde/amarillo/rojo) que refleja su estado en la cola de exportación.

Menús Contextuales (Clic Derecho)

Hacer clic derecho en diferentes áreas ofrece acciones rápidas:

  • Sobre una lista (Selección/Descartes): Ofrece acciones para mover, restaurar, eliminar o localizar la imagen.
  • Sobre la tabla de exportación: Proporciona atajos para las acciones más comunes como asignar, ignorar o quitar de la cola.
