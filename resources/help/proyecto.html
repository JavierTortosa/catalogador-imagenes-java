Modo Proyecto

El Modo Proyecto es el centro de operaciones para gestionar una selección específica de imágenes. Mientras que el Modo Visualizador es para explorar toda tu colección, el Modo Proyecto te permite concentrarte en un subconjunto de imágenes que has marcado previamente, para luego etiquetarlas, organizarlas y finalmente exportarlas.

Flujo de Trabajo Típico

  1. Marcar Imágenes: En el Modo Visualizador, usa la barra espaciadora para marcar las imágenes que te interesan.
  2. Entrar al Modo Proyecto: Haz clic en el botón "Modo Proyecto" para acceder a este espacio de trabajo dedicado.
  3. Organizar y Refinar: Utiliza las listas de "Selección Actual" y "Descartes" para organizar tu colección.
  4. Guardar tu Progreso: Guarda tu selección en un archivo .prj para poder continuar más tarde.
  5. Preparar la Exportación: Usa el potente Panel de Exportación para verificar y preparar tus imágenes y sus archivos asociados.
  6. Exportar: Copia todos los archivos listos a una carpeta de destino con un solo clic.
