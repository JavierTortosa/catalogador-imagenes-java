Modo Proyecto

El Modo Proyecto es el centro de operaciones para gestionar una selección específica de imágenes. Mientras que el Modo Visualizador es para explorar toda tu colección, el Modo Proyecto te permite concentrarte en un subconjunto de imágenes que has marcado previamente, para luego etiquetarlas y exportarlas.

Conceptos Fundamentales

El flujo de trabajo del Modo Proyecto gira en torno a tres áreas principales:

1. Selección Actual

Esta es tu lista de trabajo principal. Contiene todas las imágenes que has marcado con la barra espaciadora en el Modo Visualizador. Las imágenes en esta lista son las candidatas para ser etiquetadas y exportadas. Es el punto de partida de cualquier proyecto.

2. Descartes

La lista de Descartes funciona como un área de "apartados" o "en espera". Si mientras revisas tu "Selección Actual" decides que una imagen no es adecuada para este proyecto en particular, pero no quieres eliminarla por completo, la puedes mover a Descartes. Las imágenes en esta lista pueden ser restauradas a la "Selección Actual" o eliminadas permanentemente del proyecto en cualquier momento.

3. Pestaña de Exportación

Esta no es una lista que gestiones manualmente, sino una vista de preparación para la exportación. Cuando haces clic en la pestaña "Exportar", la aplicación toma todas las imágenes de tu "Selección Actual" y genera una cola, comprobando el estado de cada una. Aquí es donde gestionas los detalles finales antes de copiar los archivos a una carpeta de destino.

Para una descripción detallada de cada botón y pestaña, consulta la sección Controles del Modo Proyecto.
